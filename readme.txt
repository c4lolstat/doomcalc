To run:
1. download NodeJS  - https://nodejs.org/download/
2. install it
3. start commandline
4. go to application folder
5. run: Doom.bat
6. open browser
7. go to localhost:3000/doom
8. Welcome in elohell...

If server.js fail try to install express.js

Try again.
